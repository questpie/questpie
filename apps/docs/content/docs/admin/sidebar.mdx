# Sidebar Configuration

The sidebar is configured on the **server** using the `.sidebar()` method. It defines the navigation structure with sections, collection links, global links, and custom links.

## Basic Configuration

```typescript
// src/questpie/server/cms.ts
const cms = q({ name: "my-cms" })
  .use(adminModule)
  .collections({ posts, users })
  .sidebar(({ s, c }) => s.sidebar({
    sections: [
      s.section({
        id: "content",
        title: { en: "Content" },
        icon: c.icon("ph:files"),
        items: [
          { type: "collection", collection: "posts" },
          { type: "collection", collection: "users" },
        ],
      }),
    ],
  }))
  .build({ db: { url: "..." } });
```

The `({ s, c })` callback provides:
- **`s`** — Sidebar builder helpers (`s.sidebar()`, `s.section()`)
- **`c`** — Component helpers (`c.icon("ph:icon-name")`)

## Sections

Sections group navigation items. Each section has:

- **id** — Unique identifier
- **title** — Optional header text (i18n supported)
- **icon** — Optional section icon
- **collapsed** — Start collapsed (default: `false`)
- **items** — Navigation items

```typescript
s.section({
  id: "content",
  title: { en: "Content", sk: "Obsah" },
  icon: c.icon("ph:files"),
  items: [
    { type: "collection", collection: "posts" },
    { type: "collection", collection: "pages" },
  ],
})
```

## Item Types

### Collection Item

Link to a collection list:

```typescript
{
  type: "collection",
  collection: "posts",                         // Backend collection name
  label: { en: "Blog Posts" },                 // Optional label override
  icon: c.icon("ph:article"),                  // Optional icon override
}
```

If `label` and `icon` are omitted, they default to the collection's `.admin()` config.

### Global Item

Link to a global edit page:

```typescript
{
  type: "global",
  global: "siteSettings",                      // Backend global name
  label: { en: "Site Settings" },              // Optional label override
  icon: c.icon("ph:gear"),                     // Optional icon override
}
```

### Link Item

Custom navigation link:

```typescript
{
  type: "link",
  label: { en: "Documentation" },
  href: "https://docs.example.com",
  icon: c.icon("ph:book-open"),
  external: true,                              // Opens in new tab
}
```

### Page Item

Link to a custom admin page:

```typescript
{
  type: "page",
  pageId: "analytics",
  label: { en: "Analytics" },
  icon: c.icon("ph:chart-line"),
}
```

### Divider

Visual separator:

```typescript
{ type: "divider" }
```

## Auto-Generated Sidebar

If you don't call `.sidebar()`, one is automatically generated from your collections and globals:

1. Collections are grouped by their `.admin()` `group` property
2. Globals appear in a "Globals" section
3. Items are sorted by their `.admin()` `order` property

```typescript
// These admin configs...
.admin(({ c }) => ({ label: { en: "Posts" }, group: "Content", order: 1 }))
.admin(({ c }) => ({ label: { en: "Pages" }, group: "Content", order: 2 }))
.admin(({ c }) => ({ label: { en: "Users" }, group: "Team", order: 1 }))

// ...auto-generate a sidebar with "Content" and "Team" sections
```

## Complete Example

```typescript
// src/questpie/server/cms.ts
const cms = q({ name: "barbershop" })
  .use(baseInstance)
  .sidebar(({ s, c }) => s.sidebar({
    sections: [
      // Team management
      s.section({
        id: "team",
        title: { en: "Team Management", sk: "Správa tímu" },
        icon: c.icon("ph:users"),
        items: [
          { type: "collection", collection: "barbers", icon: c.icon("ph:user") },
          { type: "collection", collection: "services", icon: c.icon("ph:scissors") },
        ],
      }),

      // Bookings
      s.section({
        id: "bookings",
        title: { en: "Bookings", sk: "Rezervácie" },
        icon: c.icon("ph:calendar"),
        items: [
          { type: "collection", collection: "appointments" },
          { type: "collection", collection: "reviews", icon: c.icon("ph:star") },
        ],
      }),

      // Content
      s.section({
        id: "content",
        title: { en: "Content", sk: "Obsah" },
        icon: c.icon("ph:files"),
        items: [
          { type: "collection", collection: "pages" },
          { type: "global", global: "siteSettings", icon: c.icon("ph:gear") },
        ],
      }),

      // External links
      s.section({
        id: "external",
        title: { en: "External" },
        items: [
          {
            type: "link",
            label: { en: "Documentation" },
            href: "https://questpie.dev/docs",
            icon: c.icon("ph:book-open"),
            external: true,
          },
          { type: "divider" },
          {
            type: "link",
            label: { en: "Visit Site" },
            href: "https://example.com",
            icon: c.icon("ph:arrow-square-out"),
            external: true,
          },
        ],
      }),
    ],
  }))
  .build({ ... });
```

## Icons

Use `c.icon("ph:icon-name")` for all sidebar icons. Icons use the [Phosphor](https://phosphoricons.com/) icon set via `@iconify/react`:

```typescript
c.icon("ph:users")          // Regular weight
c.icon("ph:users-fill")     // Fill variant
c.icon("ph:users-bold")     // Bold variant
c.icon("ph:users-duotone")  // Duotone variant
```
