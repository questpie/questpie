# TanStack Barbershop Server Tour

## Configuration

- `examples/tanstack-barbershop/src/questpie/server/questpie.config.ts` — runtime config (db, adapters)
- `examples/tanstack-barbershop/src/questpie/server/modules.ts` — module registration
- `examples/tanstack-barbershop/src/questpie/server/.generated/index.ts` — codegen output

Shows:

- File convention structure with convention files
- Collections and globals in their respective directories
- Dashboard/sidebar/branding via convention files
- Functions and jobs auto-discovered from directories

## Field builder patterns

- reactive compute in `pages.ts`
- conditional visibility in `appointments.ts`
- nested object and array patterns in `barbers.ts`
