# TanStack Barbershop Functions Tour

## Function Definitions

Functions are defined using standalone `fn()` from `"questpie"` — no `rpc()` builder needed.

## Function files

- `examples/tanstack-barbershop/src/questpie/server/functions/booking.ts`
- `examples/tanstack-barbershop/src/questpie/server/functions/index.ts`

Patterns shown:

- schema validation with Zod
- app API usage inside handlers
- booking slot computation logic
- analytics-style aggregation

## Automatic routing

- `examples/tanstack-barbershop/src/questpie/server/app.ts`

Functions are discovered automatically from the `functions/` directory — no manual router composition needed.
